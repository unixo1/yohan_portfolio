소개

웹 페이지 제작과 온라인 사이트 운영에 관심이 있어 HTML과 CSS를 중심으로 학습하며 포트폴리오를 제작하고 있습니다. 아직 실무 경험은 없지만, JavaScript 기능 구현과 사이트 관리에 필요한 기본적인 지식을 학습하며 점차 실무 역량을 키워가고 있습니다.

새로운 기술을 배우는 것을 즐기고, 어려운 문제에도 적극적으로 도전하며 빠르게 적응할 자신이 있습니다. 앞으로 포토샵과 일러스트레이터 등 디자인 툴도 학습하여 상품 페이지 제작과 관리에 필요한 역량을 확장하고, 팀원과 협력하여 EC사이트 운영 업무에 실질적인 기여를 하고 싶습니다.

기술

  • HTML / CSS: 기본 페이지 제작 및 레이아웃 편집 가능
  • JavaScript: 푸터 자동 연도 변경, 스크롤 시 콘텐츠 표시 기능 구현 (AI 학습 도구와 유튜브, 인터넷 자료를 참고하여 학습 및 적용)
  • 디자인 툴: 포토샵·일러스트레이터 경험 없음, 업무 필요 시 학습 예정
  • Git, GitHub

학습

  • 웹 페이지 제작 연습: 상품 페이지를 염두에 둔 레이아웃 설계
  • JavaScript 기능 학습:
    • 푸터 연도 자동 변경 → 매년 수동 수정 필요 없도록 구현, 유지보수 효율 향상
    • 스크롤 시 콘텐츠 자연스러운 표시 → 사용자 경험(UX) 향상
  • AI 도구 활용: 기능 구현 방법 학습 및 실제 적용 경험

디자인 툴 사용 경험은 없지만, 업무에 필요한 이미지 편집·배너 제작 능력을 향후 빠르게 학습할 예정입니다. 사이트 관리와 운영 업무에 빠르게 적응하고 기여하고 싶습니다.
